## Installation

First you will need to clone this repository:

```bash
git clone https://github.com/Tomcat-42/dotfiles
```

or using **gh-cli**

```bash
gh repo clone Tomcat-42/dotfiles
```

and install it (needs to have [chezmoi](https://www.chezmoi.io) installed)
